# Snake Game PWA

Un jeu Snake responsive transformé en Progressive Web App (PWA).

## Fonctionnalités PWA

- **Installation** : Peut être installé comme une application native sur mobile et desktop
- **Offline** : Fonctionne hors ligne grâce au service worker
- **Responsive** : S'adapte à toutes les tailles d'écran et orientations
- **Thème** : Utilise un thème sombre avec couleur d'accent violet

## Comment jouer

- Utilisez les flèches directionnelles ou glissez sur l'écran tactile
- Mangez les pommes pour gagner des points
- Évitez les murs et votre propre queue

## Installation

1. Ouvrez le jeu dans un navigateur compatible PWA (Chrome, Firefox, Safari, Edge)
2. Cliquez sur le bouton "Installer" ou "Ajouter à l'écran d'accueil"
3. Lancez l'application depuis votre écran d'accueil

## Technologies utilisées

- HTML5 Canvas pour le rendu du jeu
- JavaScript ES6+ pour la logique du jeu
- Service Worker pour le cache offline
- Web App Manifest pour l'installation PWA
